Intro

What Is Python

Python is a high-level programming language. It was released in 1991 and is popular for various applications. It has a simple syntax and a large library. Moreover, it is open source.



Data Types

Python is a dynamic programming language that supports assigning variables with any data type. The main data types in Python include numbers, strings, boolean values, lists, tuples, sets, and dictionaries. Understanding these types is important as they dictate how data is stored and processed. Python also offers type conversion functions for when it's necessary to convert between types.

Data Structures


1. Lists: Ordered, mutable collections that can hold items of different types and allow for resizing. Denoted by [ ].

2. Tuples: Similar to lists, but immutable. Used to store unchangeable values. Denoted by ( ).

3. Sets: Unordered collections of unique elements. Created with { } or set(). Useful for removing duplicates and performing set operations.

4. Dictionaries: Key-value pairs for quick value access based on keys. Denoted by { } and "key: value" syntax.

5. Strings: Immutable sequences of characters for textual manipulation. Accessed using indexing and slicing.



Control Flow

Python Control Flow allows you to control execution using conditional statements and loops. if, if-else, and if-elif-else execute code based on conditions. for and while iterate or repeat code. break ends loops early, continue skips iterations, and pass does nothing. Nested control flow combines structures. Control flow is essential in programming for decision-making and repetition, creating dynamic programs.
